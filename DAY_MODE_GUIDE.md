# 白天模式使用指南

## 🌞 白天模式功能

### 📋 功能概述
为智慧换热站SCADA系统添加了完整的白天/深色主题切换功能，提供更好的用户体验和适应不同环境需求。

### 🎨 主题特性

#### 深色模式（默认）
- **背景**: 深蓝色调 (#020617)
- **面板**: 半透明深色 (#0f172a)
- **文字**: 白色系
- **网格**: 暗色点阵
- **适用场景**: 夜间监控、室内控制室

#### 白天模式
- **背景**: 浅灰白色 (#f8fafc)
- **面板**: 纯白色 (#ffffff)
- **文字**: 深色系 (#1e293b)
- **网格**: 浅色点阵
- **适用场景**: 白天操作、明亮环境

### 🔄 切换方式

#### 1. 主题切换按钮
- **位置**: 页面右上角
- **样式**: 圆形太阳图标按钮
- **交互**: 点击切换主题
- **状态**: 记忆用户选择

#### 2. 自动记忆
- **存储**: localStorage本地存储
- **持久化**: 页面刷新后保持主题选择
- **初始化**: 页面加载时自动应用保存的主题

### 🛠️ 技术实现

#### 文件结构
```
smart_heat/
├── theme-config.js          # 主题配置系统
├── smart_heating_station_terminal.html  # 主页面
├── DAY_MODE_GUIDE.md        # 使用指南
└── LOCAL_RESOURCES_SUMMARY.md  # 资源清单
```

#### 核心组件
1. **ThemeConfig对象**: 统一主题管理
2. **CSS类切换**: .light-mode / .dark-mode
3. **CSS变量**: 动态颜色更新
4. **事件系统**: themeChanged事件监听

#### 使用方法
```javascript
// 切换主题
ThemeConfig.toggle();

// 应用特定主题
ThemeConfig.apply('light');
ThemeConfig.apply('dark');

// 获取当前主题
const currentTheme = ThemeConfig.getCurrent();

// 监听主题变化
window.addEventListener('themeChanged', (e) => {
  console.log('主题已切换到:', e.detail.theme);
});
```

### 🔧 自定义配置

#### 修改主题颜色
编辑 `theme-config.js` 中的颜色配置：
```javascript
light: {
  background: '#your-bg-color',
  panel: '#your-panel-color',
  // ... 其他颜色
}
```

#### 添加新主题
```javascript

  background: '#1e3a8a',

  // ... 完整配置
```
